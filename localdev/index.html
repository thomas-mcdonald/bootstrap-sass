Collapsible Sidebar

We added a fixed-scrollable-collapsible feature, when it collapse, the sub-menu appear beside the sidebar.

Highlighted Items

Just add ".active" class in the first or sub level items and they will show open by default.

Auto Adjust

Sidebar adjusts automatically when window is resized, and also creates a scroll if there is no enough vertical space.

Fixed Sidebar

To set the sidebar fixed just add ".fixed-menu" to the "#cl-wrapper" element.

Collapsed Sidebar

To collapse the sidebar you can click on "Collapse Button" located in the bottom of the sidebar. If you want the sidebar collapsed by default just add ".sb-collapsed" to the "#cl-wrapper" element.
